Led the complete end-to-end upgrade of the ORMB (Oracle Revenue Management and Billing) platform from version 2.9 to version 5.1 across 5 environments — Dev, QA, Staging, UAT, and Production. This was not a simple patch. Every environment was built from the ground up: WebLogic server configuration, SSL/TLS certificate management, secure access controls, CI/CD pipeline design for automated deployments across all stages, and migration of 3 critical enterprise integrations (EFX, BO, Infogix) with zero data loss. The upgrade also included cost optimization by replacing F5/AIX load balancers with Nginx-based load balancing on Linux. Every environment was delivered production-ready with full security compliance.
